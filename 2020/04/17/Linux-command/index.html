Linux 命令

cd

切换当前工作目录至 dirName(目录参数)。

语法

$ cd [dirName]
  • dirName：要切换的目标目录。
  • 若省略 dirName，则变换至使用者的 home 目录
  • 另外，”~” 也表示 home 目录，”.” 表示目前所在的目录，”..” 则表示目前目录位置的上一层目录。

pwd

显示当前所在的工作目录。

语法

$ pwd

ls

显示指定工作目录下之内容（列出目前工作目录所含之文件及子目录)。它有很多参数可用。比如：

$ ls
$ ls -a
$ ls -l
$ ls -all
$ ls fileName -l
